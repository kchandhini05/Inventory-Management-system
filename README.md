# Inventory Management System

A simple Inventory Management System built using Java and MySQL.
This project helps manage products, monitor stock levels, and perform add, update, and delete operations.

## Features
- User Login and Logout
- Dashboard
- Add Product
- Update Product
- Delete Product
- Inventory Tracking

## Technologies Used
- Java
- MySQL
- RS2XML JAR
- MySQL Connector/J
- Apache Tomcat
- HTML
- CSS


## How to Run
1. Import the project into Eclipse
2. Configure Apache Tomcat server
3. Create the database using the SQL file
4. Update database credentials
5. Run the project
